# 🛒 Amazon Clone (UI Only)

A static clone of the Amazon homepage built using **HTML** and **CSS**. This project replicates the design and layout of the official Amazon website, focusing purely on the frontend interface.

## 🌟 Features

- Fully responsive homepage layout
- Amazon-style navigation bar
- Product sections with images and hover effects
- Footer with multiple sections
- Mobile-friendly design

## 🛠️ Technologies Used

- HTML5
- CSS3
- (Optional) Font Awesome / Google Fonts
